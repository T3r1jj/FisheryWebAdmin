Fishery Web Admin

https://github.com/T3r1jj/FisheryWebAdmin
The application provides an admin access to the Fishery services. It allows data management by communication with CRUD services, integration with Knowledge Base (find data -> edit -> add to CRUD) and with business logic module. Aside from that the dashboard displays some basic stats about the services.
